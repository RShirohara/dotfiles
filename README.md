# `io.github.rshirohara.dotfiles`

RShirohara's dotfiles managed by [chezmoi](https://www.chezmoi.io).

## Supported environments

- macOS
  - Apps:
    - CLI Applications (managed by [Homebrew](https://brew.sh))
    - GUI Applications (managed by [Homebrew](https://brew.sh))
- Linux
  - Distributions:
    - Red Hat
  - Apps:
    - CLI Applications (managed by [Homebrew](https://brew.sh))
